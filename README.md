# Shop App API

1. Clonar repo

2. Run ```yarn install```

3. Clone ```.env.example``` to ```.env```

4. Change enviroments

5. Run database

```
docker compose up -d
```

6. Run App in Dev mode

```
yarn start:dev
```

7. Run Seed (Demo data)
```
http://localhost:3000/seed
```
